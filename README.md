# Organic & Seasonal E-Commerce Web Application

## 🌱 Project Overview

An advanced e-commerce platform connecting organic farmers directly with consumers. The platform promotes seasonal eating, sustainable agriculture, and transparent food sourcing.

## ✨ Features

- **User Authentication** - Secure login/registration with role management (Customer/Farmer)
- **Product Browsing** - Filter by season, category, price, and organic certification
- **Shopping Cart** - Add/remove items, update quantities
- **Checkout System** - Secure order placement with multiple payment options
- **User Dashboard** - Order history, wishlist, and product management for farmers
- **Seasonal Awareness** - Products marked with seasonal badges and filters
- **Responsive Design** - Fully responsive across all devices
- **Farm-to-Consumer** - Direct connection with local farmers

## 🛠️ Technologies Used

- **Frontend**: React.js, React Router, Bootstrap 5
- **State Management**: React Context API
- **Styling**: CSS3, Bootstrap
- **Icons**: React Icons
- **Notifications**: React Toastify
- **HTTP Client**: Axios
